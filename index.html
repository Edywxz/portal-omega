Visao institucional

Um portal desenhado para informar, orientar e dar escala ao ecossistema Omega.

A estrutura segue o escopo original do projeto: pagina inicial institucional, catalogo de cursos, noticias, eventos, contato e base para futuras integracoes administrativas ou CMS.

Arquitetura clara

Separacao entre HTML semantico, CSS reutilizavel e JavaScript leve para facilitar manutencao.
